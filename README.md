# Housewarming Gift Registry - Frontend

A responsive web application for managing a housewarming gift registry built with Angular, PrimeNG, and Tailwind CSS.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   ng serve
   ```

3. Open `http://localhost:4200` in your browser

## Features

- Responsive design for mobile and desktop
- View all gifts with images
- Mark gifts as purchased
- Add new gifts to the registry
- Beautiful UI with PrimeNG components and Tailwind CSS
